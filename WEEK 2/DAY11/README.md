# Day 11 - School Management System (OOP Project)

## 📌 What I Built
Completed a School Management System using Object-Oriented Programming (OOP) in Python.

## 📚 Concepts Practiced
- Classes & Objects
- Constructors (`__init__`)
- Inheritance
- Abstract Classes (ABC)
- Method Overriding
- `super()`
- Lists of Objects
- Object Management
- Searching Records
- Deleting Records

## ✨ Features Implemented
- Created an abstract `Person` class.
- Implemented `Student` and `Teacher` classes using inheritance.
- Added students and teachers to the school.
- Displayed all student records.
- Displayed all teacher records.
- Implemented Search Student by ID.
- Implemented Search Teacher by ID.
- Implemented Delete Student by ID.
- Implemented Delete Teacher by ID.

## 🎯 Key Learning
This project helped me understand how multiple classes interact with each other, how inheritance reduces code duplication, and how objects can be managed efficiently using lists in a real-world Python application.
